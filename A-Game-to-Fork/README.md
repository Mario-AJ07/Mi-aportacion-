# Mi aportacion

He modificado el juego para que los círculos aumenten progresivamente su velocidad de caída conforme pasa el tiempo, para que haya un aumento de dificultad de manera progresiva

---

# Funcionalidades añadidas

- Velocidad de caída progresiva (los círculos caen más rápido con el tiempo).  
- Límite máximo de velocidad para mantener la jugabilidad equilibrada.  
- Ajustes en el `GameController.java` para actualizar la velocidad cada segundo.
